Detailed Description

A synchronized ordered collection object that represents the results of a query to the application's data source(s). Results may arrive in any order, and the IncrementalResult notifies observers that results have arrived, and in what position in the result when they do arrive. The IncrementalResult also supports type filtering. If a query would result in multiple types of results being returned, both the getter and notification functions of the IncremencalResult come in type-filtered variants as well.
