\subsection{Detailed Description}
A synchronized ordered collection object that represents the results of a query to the application's data source(s). Results may arrive in any order, and the \hyperlink{classcom_1_1ualberta_1_1team17_1_1datamanager_1_1_incremental_result}{Incremental\+Result} notifies observers that results have arrived, and in what position in the result when they do arrive. The \hyperlink{classcom_1_1ualberta_1_1team17_1_1datamanager_1_1_incremental_result}{Incremental\+Result} also supports type filtering. If a query would result in multiple types of results being returned, both the getter and notification functions of the Incremencal\+Result come in type-\/filtered variants as well.
